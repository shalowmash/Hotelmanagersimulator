Aimed at people interested management, this simulation will allow the user to run a hotel business, accounting for different scenarios that may occur. By using this simulator, the user gains knowledge into the day to day running of a hotel, allowing them to make an educated choice as to whether or not this field is for them.

Features:

This code will allow the user to...

  • Create unique Login to save progress
  • Assign and manage rooms
  • Collect room fees
  • Expand and upgrade the hotel!
  • Handle utility bills
